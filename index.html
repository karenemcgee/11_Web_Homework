Summary: Latitude vs X

In this project, we looked at global weather data that included information about the city's location (country, latitude, and longitude); the date the data was collection; and the humidity, maximum temperature, wind speed, and cloudiness for that date. From this data, we randomly selected 500+ cities based on their latitude and longitude in order to run a weather check on each one using successive OpenWeatherMap API calls.


Using the data collected from these 500+ cities, we utilized Matplotlib to make four individual scatterplots depicting the humidity, maximum temperature, wind speed, and cloudiness as it relates to the longitude of each city. These plots allowed us to analyze the trends and correlations observed within this data.
